# Приложение для учета расходов и доходов
# Application for tracking expenses and income
![Application](assets/screenshot_.png)
## Описание
В планах разработка удобного, простого и быстрого <br>  приложения, для контроля расходов и доходов. <br>
С возможностью просмотра и сравнения данных, <br>
полученных за год, за месяц и за все время <br>
в виде графиков
## Технологии
Бекенд микросервисы на Go <br>
Фронтенд React, TypeScrypt
